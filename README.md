# Smart-mobility-Wheelchair

## 📌 Overview
This project is an autonomous smart wheelchair using ROS 2 for navigation and control.

## 🚀 Features
- Autonomous navigation  
- LiDAR-based mapping (SLAM)  
- Voice control  
- Mobile application integration  

## 🛠️ Technologies Used
- ROS 2  
- Raspberry Pi  
- LiDAR  
- Python / C++  

## 👩‍💻 My Role
I worked on implementing the autonomous navigation system and integrating LiDAR for mapping using SLAM.
